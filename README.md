# PPT God ✨

### 让 AI 做出一套真正能交付的 PPT

> 输入主题、文档、逐字稿、Logo、产品截图和参考图，生成一套风格一致、品牌准确、产品可信的演示文稿。

---

## 🌱 品牌故事

PPT God 最早来自一次真实分享任务：我需要为一个行业论坛准备 PPT。

我试过很多工具。NotebookLM 很会理解资料，Lovart 很会提供视觉灵感，普通 AI PPT 工具也能快速出稿。但真正要交付时，问题不再是“能不能生成一页好看的图”，而是：

- Logo 不能被 AI 重新画错。
- 页面不能各自好看，却放在一起不像一套作品。
- 产品截图、界面、实物和关键卖点不能被自由发挥。
- 每一页讲什么，必须先让人看得懂、改得动、确认得了。

所以 PPT God 不是为了让 PPT 更像盲盒，而是让 AI 按照真实 PPT 的标准工作：理解内容，尊重素材，统一视觉，最后交付一份能继续修改和使用的文件。

## 🚀 怎么使用？

1. **输入内容**：给它主题、文档、逐字稿或粗略想法。
2. **上传素材**：加入 Logo、产品截图、参考图、PDF 或 PPT 模板。
3. **确认方向**：先看每页讲什么、整体长什么样。
4. **生成页面**：让它生成完整页面，而不是只给建议。
5. **继续修改**：用自然语言调整某一页或整套风格。
6. **下载 PPT**：得到可以检查、修改和交付的文件。

## ⭐ 特殊功能

| 功能 | 为什么重要 |
| --- | --- |
| 🛡️ **Logo 保真** | Logo 是品牌资产，不能让模型随意重绘 |
| 🧬 **跨页面一致** | 封面、章节页、内容页、数据页要像同一套作品 |
| 📸 **产品展示准确** | 截图、界面、实物和结构图要作为事实依据 |
| 🧭 **每页可判断** | 生成前先看到每页内容和画面方向，减少返工 |
| 💬 **修改能落页** | “第 3 页更商务一点”应该真的改变第 3 页 |
| 🎛️ **可操作界面** | 内容、素材、页面、风格和结果都能被看见和调整 |

## 🆚 和普通 AI PPT 工具有何不同？

| 普通 AI PPT 工具 | PPT God |
| --- | --- |
| 快速出一版草稿 | 面向正式交付 |
| 单页看起来不错 | 整套页面保持一致 |
| Logo 容易被画错 | Logo 作为品牌资产保真 |
| 产品图容易被自由发挥 | 产品素材作为事实依据 |
| 聊天回答和页面修改脱节 | 用户意见尽量落到具体页面 |
| 生成前很难判断内容 | 先确认内容与方向，再生成 |

## 🧠 核心信念

好 PPT 的难点，不是把文字塞进模板，而是让内容判断、视觉秩序、品牌准确性和产品可信度在几十页里保持同一个标准。

PPT God 想做的，就是让 AI 从“帮你想一想”走向“帮你做出一份真正能用的 PPT”。
